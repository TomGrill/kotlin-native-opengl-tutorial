# OpenGL tutorial written in Kotlin Native

The tutorial could be found [here](http://opengl-tutorial.org).

## Requirementes

- [GLFW](http://www.glfw.org)
- [GLEW](http://glew.sourceforge.net)

**MacOS**

```
brew install glfw glew
```

## Running

```
gradle build
gradle run
```
